# Disable your ad blocker for this site

If you are seeing a blocking prompt, one of your extensions is likely hiding required calculator scripts.

## Chrome + AdBlock / Adblock Plus

1. Open the extension from the toolbar.
2. Select **Pause on this site** or **Don't run on pages on this site**.
3. Reload the page.
4. Click **Retry** in the prompt.

## Chrome + uBlock Origin

1. Open uBlock Origin.
2. Click the blue power button for this site.
3. Reload this page.
4. Click **Retry**.

## Ghostery

1. Open Ghostery.
2. Trust this site or pause blocking for this site.
3. Reload and retry.

## Firefox

1. Open your blocker add-on settings.
2. Add this domain to the allow-list.
3. Reload and retry.
